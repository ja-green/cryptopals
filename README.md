## Installing

These solutions are written in Go and therefore the Go tool is required for compiling the code

Clone from Github:

```bash
$ git clone git@github.com:ja-green/cryptopals.git
```

Compile using Go:

```bash
$ go build
```

## Usage

```bash
$ cryptopals [set number] [challenge number]
```
